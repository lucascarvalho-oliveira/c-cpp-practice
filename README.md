# Repositório de Estudos em C e C++
Este repositório reúne os códigos desenvolvidos durante meus estudos em C e C++.

O objetivo é registrar minha evolução prática, consolidar fundamentos e construir uma base sólida em programação estruturada e orientada a objetos.

## Objetivo
- Praticar lógica de programação
- Dominar ponteiros, memória e estruturas de dados
- Entender profundamente compilação e execução
- Evoluir da programação estruturada (C) para conceitos de POO (C++)

Este repositório funciona como meu laboratório pessoal de aprendizado.

## Evolução
Este repositório será constantemente atualizado conforme avanço nos estudos.

A ideia não é apenas guardar códigos, mas acompanhar minha evolução técnica ao longo do tempo.
